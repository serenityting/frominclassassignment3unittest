### **API & Reference**

#### `my_adder(a, b)`
- **Description:** Adds two numbers and returns the sum.
- **Parameters:**
  - `a` (number): The first number.
  - `b` (number): The second number.
- **Returns:** `number` — The sum of `a` and `b`.

#### `my_thermostat(set_temp)`
- **Description:** Sets the thermostat to the specified temperature and returns the current temperature.
- **Parameters:**
  - `set_temp` (number): Desired temperature setting.
- **Returns:** `number` — Current temperature after setting.

#### `have_digits(number, digits)`
- **Description:** Checks if the number contains all specified digits.
- **Parameters:**
  - `number` (integer): The number to check.
  - `digits` (array of integers): Digits to look for.
- **Returns:** `boolean` — `true` if all digits are present, `false` otherwise.

#### `area_of_rectangle(length, width)`
- **Description:** Calculates the area of a rectangle.
- **Parameters:**
  - `length` (number): The length of the rectangle.
  - `width` (number): The width of the rectangle.
- **Returns:** `number` — The calculated area.

#### `perimeter_of_rectangle(length, width)`
- **Description:** Calculates the perimeter of a rectangle.
- **Parameters:**
  - `length` (number): The length of the rectangle.
  - `width` (number): The width of the rectangle.
- **Returns:** `number` — The perimeter length.
